# Echo (صدى)

Echo ist eine private, beruhigende und wunderschön gestaltete Erinnerungs-Tagebuch-Anwendung.

Bei Echo glauben wir, dass jeder einen ruhigen, sicheren Ort für seine Erinnerungen verdient. Mit moderner Ästhetik, Glassmorphismus und einem Fokus auf Barrierefreiheit bietet Echo eine sichere Umgebung für Nutzer, um ihr Leben ohne Ablenkungen oder Lärm zu dokumentieren.

## Kernwerte
- **Datenschutz:** Deine Erinnerungen gehören dir allein.
- **Barrierefreiheit:** Entwickelt, um inklusiv und für jeden einfach zu bedienen zu sein.
- **Ästhetik:** Ein ruhiges Erlebnis im Dark Mode, das die Augenbelastung reduziert und ein Gefühl der Ruhe vermittelt.
