# Echo (صدى)

O Echo é um aplicativo de diário de memórias privado, reconfortante e lindamente projetado.

No Echo, acreditamos que todos merecem um lugar tranquilo e seguro para guardar suas memórias. Construído com uma estética moderna, glassmorphism e foco na acessibilidade, o Echo oferece um ambiente seguro para os usuários documentarem suas vidas sem distrações ou ruídos.

## Valores Essenciais
- **Privacidade:** Suas memórias são apenas suas.
- **Acessibilidade:** Projetado para ser inclusivo e fácil de usar para todos.
- **Estética:** Uma experiência calma e com tema escuro que reduz o cansaço visual e proporciona uma sensação de tranquilidade.
