# Echo (صدى)

Echo es una aplicación de diario de recuerdos privada, reconfortante y con un diseño hermoso.

En Echo, creemos que todos merecen un lugar tranquilo y seguro para guardar sus recuerdos. Creado con una estética moderna, diseño de cristal (glassmorphism) y un enfoque en la accesibilidad, Echo proporciona un entorno seguro para que los usuarios documenten sus vidas sin distracciones ni ruido.

## Valores fundamentales
- **Privacidad:** Tus recuerdos son solo tuyos.
- **Accesibilidad:** Diseñado para ser inclusivo y fácil de usar para todos.
- **Estética:** Una experiencia tranquila y de tema oscuro que reduce la fatiga visual y proporciona una sensación de tranquilidad.
