# Echo (صدى)

Echo is a private, comforting, and beautifully designed memory journal application.

At Echo, we believe everyone deserves a quiet, safe place to store their memories. Built with modern aesthetics, glassmorphism, and a focus on accessibility, Echo provides a secure environment for users to document their lives without distractions or noise.

## Core Values
- **Privacy:** Your memories are yours alone.
- **Accessibility:** Designed to be inclusive and easy to use for everyone.
- **Aesthetics:** A calm, dark-themed experience that reduces eye strain and provides a sense of tranquility.
