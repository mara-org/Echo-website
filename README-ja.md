# Echo (صدى)

Echoは、プライベートで心地よく、美しくデザインされた思い出のジャーナルアプリです。

Echoでは、誰もが思い出を保存するための静かで安全な場所を持つべきだと信じています。モダンな美学、グラスモーフィズム、アクセシビリティに重点を置いて構築されたEchoは、ユーザーが気晴らしやノイズなしに自分の人生を記録するための安全な環境を提供します。

## コアバリュー
- **プライバシー:** あなたの思い出はあなただけのものです。
- **アクセシビリティ:** すべての人にとって包括的で使いやすいように設計されています。
- **美学:** 目の疲れを軽減し、静けさを提供する、落ち着いたダークテーマの体験。
